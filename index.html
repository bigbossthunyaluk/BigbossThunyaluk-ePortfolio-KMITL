ผลงานที่ภาคภูมิใจ
  • ได้รับรางวัลชมเชยการประกวดโครงงานวิทยาศาสตร์ ระดับมัธยมศึกษาตอนปลาย สาขาวิทยาศาสตร์ประยุกต์

จัดโดย : กระทรวงการอุดมศึกษา วิทยาศาสตร์ วิจัยและนวัตกรรม ร่วมกับ สมาคมวิทยาศาสตร์แห่งประเทศไทยในพระบรมราชูปถัมภ์ และ คณะวิทยาศาสตร์ มหาวิทยาลัยขอนแก่น

ในงานสัปดาห์วันวิทยาศาสตร์ ส่วนภูมิภาค(ภาคตะวันออกเฉียงเหนือ) ประจำปี พ.ศ.2564 ณ คณะวิทยาศาสตร์ มหาวิทยาลัยขอนแก่น ในหัวข้อโครงงานเรื่อง : ระบบช่วยเหลืออาสาสมัครหมู่บ้านตรวจสอบ Antigen Test Kit โดยใช้ machine learning

เข้าร่วม ค่ายเยาวชนวิศวกรรมคอมพิวเตอร์
ภาคฤดูร้อนครั้งที่ 16 ( CESCa16 - OnlineCamp )

จัดโดย : สาขาวิศวกรรมคอมพิวเตอร์ คณะวิศวกรรมศาสตร์ มหาวิทยาลัยขอนแก่น

ค่ายเยาวชนวิศวกรรมคอมพิวเตอร์ ภาคฤดูร้อน หรือค่าย CESCa เนื่องจากอยู่ในช่วงของการแพร่ระบาดของ Covid-19 การจัดค่าย CESCa ครั้งนี้จึงเป็นค่ายออนไลน์ ซึ่งภายในค่ายมีการให้ความรู้ในการเขียนโปรแกรมขั้นพื้นฐาน ได้แก่ การใช้ git เบื้องต้น, Machine Learning, การสร้างเว็ปไซต์โดยใช้ภาษา HTML CSS JavaScript และการเขียนโปรแกรมควบคุมหุ่นยนต์ โดยพี่ๆ จากสาขาวิศวกรรมคอมพิวเตอร์ คณะวิศวกรรมศาสตร์ มหาวิทยาลัยขอนแก่น

เข้าร่วม การอบรมให้ความรู้เรื่องคอมพิวเตอร์เชิงควอนตัม Summer School Quantum Computing (SCQC2021)

จัดโดย : คณะวิทยาศาสตร์ มหาวิทยาลัยเทคโนโลยีพระจอมเกล้าธนบุรี ร่วมกับสาขาวิชาฟิสิกส์ คณะวิทยาศาสตร์และเทคโนโลยี มหาวิทยาลัยธรรมศาสตร์

เป็นการอบรมให้ความรู้ภาคประชาชนทั่วไป เกี่ยวกับหลักการทำงานของเครื่องคอมพิวเตอร์เชิงควอนตัม หลักการทางฟิสิกส์ ที่อยู่เบื้องหลังการทำงาน รวมถึงการนำหลักการของควอนตัมคอมพิวเตอร์ไปประยุกต์ใช้ พร้อมกับมีการอบรมภาคปฏิบัติ โดยอาจารย์จากคณะวิทยาศาสตร์ มหาวิทยาลัยเทคโนโลยีพระจอมเกล้าธนบุรี และสาขาวิชาฟิสิกส์ คณะวิทยาศาสตร์และเทคโนโลยี มหาวิทยาลัยธรรมศาสตร์

เข้าร่วม การแข่งขัน Thailand Cyber Top Talent 2021 ระดับมัธยมศึกษา

จัดโดย : สำนักงานคณะกรรมการการรักษาความมั่นคงปลอดภัยไซเบอร์แห่งชาติ (สกมช.)

เป็นการแข่งขันเกี่ยวกับความปลอดภัยทางไซเบอร์ เป็นการแข่ง Capture the Flag : CTF ในรูปแบบ Jeopardy ซึ่งต้องแข่งกันคำตอบ(flag)ที่จะซ่อนอยู่ในโจทย์ ตามหัวข้อต่างๆ โดยผมได้เข้าร่วมแข่งขันในนามทีม "hACKERMancha" ได้คะแนน 600 คะแนน ซึ่งเป็นคะแนนอันดับที่ 5 ของภาคตะวันออกเฉียงเหนือ

เข้าร่วม โครงการแข่งขัน The 2nd Kibo Robot Programming Challenge

จัดโดย : สํานักงานพัฒนาวิทยาศาสตร์และเทคโนโลยีแห่งชาติ(สวทช.)
ร่วมกับ องค์การสํารวจอวกาศญี่ปุ่น(JAXA)

เป็นโครงการแข่งขันเขียนโปรแกรมควบคุมหุ่นยนต์อวกาศ โดยรอบแรกจะเป็นการเขียนโปรแกรม(ภาษา Java)ส่งออกไฟล์เป็นรูปแบบของ APK ไปควบคุมหุ่นยนต์ Kibo Robot ใน simulator ให้ทำภารกิจตามที่กำหนด คือควบคุมหุ่นยนต์ไปตามเขตพื้นที่ที่กำหนดไว้ สแกน Qrcode ที่ติดอยู่บนผนัง airlock ยิงเลเซอร์ไปที่เป้า ถ่ายภาพผลการยิงเลเซอร์ แล้วไปรายงานผลกับนักบินอวกาศ ผมเข้าร่วมแข่งขันในนามทีม "Made in Mancha Khiri" ครับ

โปรเจค

project

  • เว็ปไซต์ระบบช่วยเหลืออาสาสมัครหมู่บ้านตรวจสอบ Antigen Test Kit โดยใช้ MachineLearning

ภาษา : HTML, CSS (Bootstrap), JavaScript

เครื่องมือ : Google Teachable Machine, Google Sheets, Github

เป็นเว็ปไซต์สำหรับให้ผู้ใช้งานทั่วไปที่เป็นผู้เดินทางมาจากพื้นที่เสี่ยงในการติดเชื้อ Covid-19 เข้ามากรอกข้อมูลพร้อมกับนำชุดตรวจ ATK มาประมวลผลผ่านหน้าเว็ปเพื่อบันทึก ผลของชุดตรวจนั้น และมีส่วนของหน้า dashboard สำหรับอาสาสมัครหมู่บ้าน(อสม.)ที่จะนำข้อมูลไปใช้งาน ระบบนี้สร้างโดยการ Training MachineLearning ผ่าน Google Teachable Machineโดยใช้รูปภาพชุดตรวจ Covid-19 แบบ ATK จะได้เป็น model สำเร็จรูปสำหรับใช้ในเว็ปไซต์ แล้วเขียนเว็ปไซต์มีหน้า Home ที่สามารถลิงค์เข้าไปที่หน้า User สำหรับผู้ใช้งานทั่วไป และหน้า Login สำหรับอสม.ลงชื่อเข้าใช้งาน ก่อนที่จะเข้าสู่หน้า Admin ที่จะมี dashboard แสดงข้อมูลที่ผู้ใช้บันทึกข้อมูล เก็บข้อมูลไว้ที่ Google Sheets ส่วนหน้าเว็ปไซต์จะเก็บไว้ที่ Github Webpage

  • กล่อง IoT รายงานสถานะอุณหภูมิ ความชื้น แรงดันไฟฟ้า ผ่านแอปพลิเคชั่น Blynk และ Line

ภาษา : C (Arduino IDE)

อุปกรณ์ : Arduino NodeMCU ESP8266, DHT11, ZMPT101B

โปรเจคนี้เป็นโครงงาน IS ระดับชั้นมัธยมศึกษาปีที่ 5 ของผมครับ ซึ่งเป็นกล่อง Iot(internet of things) ที่มีเซ็นเซอร์รับค่าอุณหภูมิ ความชื้น แรงดันไฟฟ้า ต่อเข้ากับบอร์ด Arduino (node MCU) เขียนโปรแกรมให้ต่อกับ wifi บ้าน แล้วส่งค่าเข้าไปที่แอปพลิเคชั่น Blynk และ Line โดยเขียนให้ส่งค่าอุณหภูมิและความชื้นของอากาศช่วงเช้าไปที่ Line ผ่าน bot line notify และแจ้งเตือนเมื่อค่าแรงดันไฟฟ้ามีความผิดปกติ เช่น ไฟดับ ไฟตก มีปลั๊กสำหรับควบคุมการเปิดปิดเครื่องใช้ไฟฟ้าผ่านแอปพลิเคชัน Blynk
